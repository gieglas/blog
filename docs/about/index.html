About Me

Hello, I’m Constantinos Evangelou, an interaction designer working for the Digital Services Factory (DSF) who loves to play with code and this is where I write about stuff.

I helped implement various government services, but what I really enjoy is creating reusable tools to help others create simple, accessible services that improve people’s lives.

Current projects

The gov.cy design system developer assets project

govcy-design-system-docs
